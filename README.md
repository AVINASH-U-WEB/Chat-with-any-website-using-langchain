# Chat with Any Website

A full-stack application that allows you to chat with any website using Gemini AI. Built with FastAPI (Python backend), React (Vite frontend), and LangChain.\

## Features
- Chat with content from any website URL
- Session-based conversation history
- Real-time AI responses using Gemini
- Clean and responsive UI with Tailwind CSS
- Full-stack architecture with separate frontend/backend

### Key Workflow Details:

1. **Initialization Phase**:
   - User enters website URL in frontend
   - Frontend sends `POST /init_session` to backend
   - Backend scrapes website, chunks content, generates embeddings
   - Creates vector store and new session
   - Returns session ID to frontend

2. **Chat Phase**:
   - User sends message through frontend
   - Frontend sends `POST /chat` with session ID and message
   - Backend retrieves session-specific vector store and chat history
   - LangChain generates search query based on conversation history
   - Relevant content is retrieved from vector store
   - Gemini generates response using context and conversation history
   - Response is sent back to frontend and displayed

3. **LangChain Processing**:
   - History-aware retriever maintains conversation context
   - Retrieval chain finds relevant document chunks
   - Stuff documents chain combines context with conversation
   - Gemini produces final response

4. **Session Management**:
   - Backend maintains in-memory session storage
   - Each session contains:
     - Vector store (website-specific embeddings)
     - Chat history (conversation context)
   - Session expires when server restarts
